Problema de la lata de conserva

Una empresa desea fabricar latas de conserva de forma cilndrica, con un dimetro entre \(10\) y \(20\) centmetros y una altura entre \(10\) y \(30\) centmetros, de tal forma que la lata tenga mxima capacidad con el menor coste de material posible. Se trata, por tanto, de un problema de optimizacin multiobjetivo que se puede formalizar como sigue: \[\begin{align*} \text{maximizar} \quad &\frac{\pi x_{1}^{2} x_{2}}{4} \quad (\text{Volumen de un cilindro de dimetro } x_{1} \text{ y altura } x_{2}) \\ \text{minimizar} \quad &\frac{\pi x_{1}^{2}}{2} + \pi x_{1} x_{2} \quad (\text{Superficie de un cilindro de dimetro } x_{1} \text{ y altura } x_{2}) \\ \text{sujeto a} \quad &x_{1}, x_{2} \in \mathbb{R} \\ &1 \leq x_{1}, x_{2} \leq 30 \end{align*}\]

Para abordar este problema mediante algoritmos genticos, consideramos genotipos de nmeros reales de longitud \(2N = 6\) representando soluciones en las que el dimetro de la lata es la media de los \(N = 3\) primeros genes y la altura de la lata es la media de los \(N = 3\) ltimos genes.

obten_fenotipo <- function(genotipo) {
  diametro <- mean(genotipo[seq_len(params$N)])
  altura <- mean(genotipo[-seq_len(params$N)])
  c(`Dimetro` = diametro, Altura = altura)
}

Como valor de aptitud de cada genotipo consideramos una combinacin ponderada de las dos funciones objetivo a optimizar. Podra ocurrir que los valores de una de esas funciones fueran de mayor magnitud que los de la otra funcin, quedando anulados los segundos por los primeros. Para evitar este problema normalizamos al intervalo \((0, 1)\) los valores de ambas funciones. Adems, puesto que el algoritmo gentico implementado en el paquete GA trata de maximizar el valor de aptitud, transformamos el objetivo de minimizar la superficie de la lata por el objetivo de maximizar la diferencia entre \(S\) y la superficie de la lata, donde \(S\) es la mxima superficie posible.

calcula_volumen <- function(fenotipo) {
  diametro <- fenotipo[[1]]
  altura <- fenotipo[[2]]
  pi * diametro^2 * altura / 4
}

maximo_volumen <- calcula_volumen(c(20, 30))

calcula_superficie <- function(fenotipo) {
  diametro <- fenotipo[[1]]
  altura <- fenotipo[[2]]
  pi * diametro^2 / 2 + pi * diametro * altura
}

maxima_superficie <- calcula_superficie(c(20, 30))

funcion_fitness <- function(genotipo) {
  fenotipo <- obten_fenotipo(genotipo)
  volumen <- calcula_volumen(fenotipo)
  volumen_normalizado <- volumen / maximo_volumen
  superficie <- calcula_superficie(fenotipo)
  superficie_normalizada <- superficie / maxima_superficie
  params$peso_volumen * volumen_normalizado +
    params$peso_superficie * (1 - superficie_normalizada)
}

Estamos ya entonces en condiciones de poder abordar el problema mediante algoritmos genticos. Hay que tener en cuenta que un problema de optimizacin multiobjetivo no tiene soluciones ptimas globales, por lo que la solucin encontrada por el algoritmo depende en gran medida de los pesos considerados para cada funcin objetivo, que en este caso son \(1\) para el volumen de la lata y \(1\) para la superficie de la lata.

library(GA)
library(purrr)

ultima_generacion <- 
  ga(type = "real-valued",
     lower = rep(10, times = 2 * params$N), # Mnimo valor posible para cada gen
     upper = rep(c(20, 30), each = params$N), # Mximo valor posible para cada gen
     fitness = funcion_fitness,
     popSize = 1000,
     monitor = TRUE)
